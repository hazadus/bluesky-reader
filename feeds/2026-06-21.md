💬 Python Tip #172 (of 365):

Know your object introspection tools 🧵

When you're in an interactive Python environment remember that these 4 tools can be helpful for introspecting the state of your Python process:

1. type
2. help
3. dir
4. vars

#Python #DailyPythonTip

[Открыть пост](https://bsky.app/profile/trey.io/post/3mosxgw4lzi2r)

----
**🕒 18:04 👤 @trey.io (Trey Hunner)**
💬 type shows the class of an object ("class" and "type" are synonyms).

If you have a mystery object, type can tell you what it is:

>>> type(letter_counts)

[Открыть пост](https://bsky.app/profile/trey.io/post/3mosxgwa2cc2h)

**🕒 18:04 👤 @trey.io (Trey Hunner)**

>>> help(letter_counts)

class Counter(builtins.dict)
 |  Counter(iterable=None, /, **kwds)
 |  ...

[Открыть пост](https://bsky.app/profile/trey.io/post/3mosxgwcskz2q)

**🕒 18:04 👤 @trey.io (Trey Hunner)**
💬 vars returns a dictionary of an object's instance attributes (the attributes that live DIRECTLY on an object instead of on its class):

>>> vars(args)
{'file': PosixPath('frankenstein.txt')}

Note: vars only works for objects that use __dict__ (most objects that aren't the built-in data types).

**🕒 18:04 👤 @trey.io (Trey Hunner)**
💬 You can also call dir and vars WITHOUT arguments.

dir() shows all local variables:

>>> dir()
['Counter', ..., 'letter_counts']

**🕒 18:04 👤 @trey.io (Trey Hunner)**
💬 And vars() returns a dictionary mapping all local variables to their values:

>>> vars()
{'__name__': '__main__', .., 'Counter': , 'letter_counts': Counter()}

In PDB you can use "pp dir()" & "pp vars()" to pretty-print the list/dict that these return.
