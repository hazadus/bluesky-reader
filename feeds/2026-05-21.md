**🕒 05:07 👤 @trey.io (Trey Hunner)**
💬 In Python, instead of writing this:

if hasattr(value, "__len__"):
    length = len(value)
else:
    length = None

We'd usually write this:

try:
    length = len(value)
except TypeError:
    length = None

It's easiest to find out whether something has a length by just trying it out.

💬 Don't be afraid of using exception handling instead of if-else checks.

Handling an exception does require a bit of extra machinery, so handling many exceptions in a tight loop could slow your code down.

But handling an occasional exception shouldn't be a performance concern.
