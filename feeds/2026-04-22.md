**🕒 18:04 👤 @trey.io (Trey Hunner)**
💬 Python Tip #112 (of 365):

Don't overlook keyword-only arguments 🧵

Instead of:

def __init__(self, *args, **kwargs):
    self.‍name = kwargs.pop("name", None)
    ...

Do:

def __init__(self, *args, name=None, **kwargs):
    self.‍name = name
    ...

#Python #DailyPythonTip

**🕒 18:04 👤 @trey.io (Trey Hunner)**
💬 Any argument that comes after * in a function definition's arguments MUST be specified as a keyword argument. So "name" can be optionally provided and it defaults to None.

Many folks got that first idiom engrained in their Python muscle memory before Python 3.5 added the newer syntax.
