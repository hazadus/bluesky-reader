💬 Python Tip #71 (of 365):

Print non-standard output to standard error 🧵

Python's print function writes to the "standard output" stream by default.

But terminals have a second output stream that also writes to the screen: standard error.

#Python #DailyPythonTip

**🕒 18:04 👤 @trey.io (Trey Hunner)**
💬 Standard error is most often used for:

- error messages
- diagnostic output
- output that shouldn't be mixed in with the rest of the program's output

[Открыть пост](https://bsky.app/profile/trey.io/post/3mguye3dlnq27)

**🕒 18:04 👤 @trey.io (Trey Hunner)**
💬 Why print to standard error at all?

Well, when if the user of your program redirects the standard output stream to a file or pipes it to another program, your diagnostic output shouldn't ALSO be redirected.

So it's best to print non-standard output to standard error.

More: https://pym.dev/standard-error/
