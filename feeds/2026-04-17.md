**🕒 18:04 👤 @trey.io (Trey Hunner)**
💬 Python Tip #107 (of 365):

Use the walrus operator to simplify "while" loops. 🧵

In particular, use it to simplify a "while" loop that performs the same operation both BEFORE the loop and WITHIN the loop for the sake of computing the "while" condition.

#Python #DailyPythonTip

[Открыть пост](https://bsky.app/profile/trey.io/post/3mjpj5p5qiy2r)

----
**🕒 18:04 👤 @trey.io (Trey Hunner)**
💬 This generator function reads a file in chunks:

def read_in_chunks(path):
    with open(path, mode="rb") as file:
        chunk = file.‍read(8192)
        while chunk:
            yield chunk
            chunk = file.‍read(8192)

Notice the double "chunk = " lines: one before and one in the loop.

[Открыть пост](https://bsky.app/profile/trey.io/post/3mjpj5pazsw23)

----
**🕒 18:04 👤 @trey.io (Trey Hunner)**
💬 This is a somewhat common pattern with "while" loops.

We could simplify that "while" loop with an assignment expression:

def read_in_chunks(path):
    with open(path, mode="rb") as file:
        while chunk := file.‍read(8192):
            yield chunk

No more duplicated "file.‍read(8192)" call! 🎉

[Открыть пост](https://bsky.app/profile/trey.io/post/3mjpj5pdnxi2r)

----
**🕒 18:04 👤 @trey.io (Trey Hunner)**
💬 Assignment expressions are basically assignment statements that can be embedded within another Python expression.

The ":=" operator that powers assignment expressions is often called "the walrus operator" because it looks like a walrus laying on its side. See the eyes and the tusks?
