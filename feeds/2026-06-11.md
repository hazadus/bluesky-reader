💬 Python Tip #161 (of 365):

To flatten an iterable-of-iterables, use chain.from_iterable. 🧵

#Python #DailyPythonTip

**🕒 00:21 👤 @trey.io (Trey Hunner)**
💬 Got a list of lists?

>>> groups = [["Hong", "Ryan"], ["Anthony", "Willa"], ["Margaret", "Adrian"]]

Flatten it with a nested comprehension:

names = [name for group in groups for name in group]

Or use chain.from_iterable:

from itertools import chain
names = list(chain.from_iterable(groups))

[Открыть пост](https://bsky.app/profile/trey.io/post/3mnxxfbeo6k2p)

----
**🕒 00:21 👤 @trey.io (Trey Hunner)**
💬 Why chain.from_iterable?

1. It's lazy: it doesn't create a new list in memory until you ask it to, like a generator expression
2. It's more succinct than an equivalent comprehension

[Открыть пост](https://bsky.app/profile/trey.io/post/3mnxxfbep5s2p)

----
**🕒 00:21 👤 @trey.io (Trey Hunner)**
💬 If you're just looping over the result, you can embrace laziness by skipping the "list(...)" call:

for name in chain.from_iterable(groups):
    print(name)

It also works with ANY iterable-of-iterables. You could flatten a generator of tuples, a file of lines split into words, etc.

----
**🕒 00:21 👤 @trey.io (Trey Hunner)**
💬 Whatever you do, don't use sum for flattening:

names = sum(groups, [])

That concatenates each sub-list to the flattened version of every list before it, making for a brand new (potentially very large) list for every sub-list.

See how slow it is for lists with thousands of sub-lists 👇

**🕒 00:21 👤 @trey.io (Trey Hunner)**
💬 When Python 3.15 is released, you'll also be able to use * for flattening with comprehensions & generator expressions:

names = [*group for group in groups]

I find that about as readable as chain.from_iterable.

But until I'm on 3.15, I'd reach for chain.from_iterable.

For more read this article 👇

![import dc from 'node:diagnostics_channel'
import { trace, SpanStatusCode } from '@opentelemetry/api'

export default defineNitroPlugin(() => {
  const tracer = trace.getTracer('nuxt')
  for (const name of ['nuxt.render', 'nuxt.island', 'nuxt.data', 'nuxt.plugin']) {
    dc.subscribe(`tracing:${name}:start`, ctx => {
      ctx._span = tracer.startSpan(name, { attributes: deriveAttrs(name, ctx) })
    })
    dc.subscribe(`tracing:${name}:error`, ctx => {
      ctx._span?.recordException(ctx.error)
      ctx._span?.setStatus({ code: SpanStatusCode.ERROR })
    })
    dc.subscribe(`tracing:${name}:asyncEnd`, ctx => ctx._span?.end())
  }
})](https://cdn.bsky.app/img/feed_fullsize/plain/did:plc:jbeaa5kdaladzwq3r7f5xgwe/bafkreih7lhfpmdiwiwkm33ux3fin2rh6tfg5jteux4jlzineokumakonmm)
