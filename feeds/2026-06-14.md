💬 Python's sets support set arithmetic using operators.

>>> a = {1, 2, 3, 4, 7}
>>> b = {1, 3, 5, 7, 9}

Union:

>>> a | b
{1, 2, 3, 4, 5, 7, 9}

Intersection:

>>> a & b
{1, 3, 7}

Asymmetric difference:

>>> a - b
{2, 4}

Symmetric difference:

>>> a ^ b
{2, 4, 5, 9}

💬 Python Tip #165 (of 365):

You can make a set of sets by using frozenset. 🧵

Sets can only contain hashable objects.

So you can't make a set-of-sets with regular sets:

>>> groups = {{"a", "b"}, {"c", "d"}}
...
TypeError: unhashable type: 'set'

#Python #DailyPythonTip

[Открыть пост](https://bsky.app/profile/trey.io/post/3mobvsqtcnk2p)

**🕒 23:19 👤 @trey.io (Trey Hunner)**
💬 If you need a set of sets, use frozenset:

>>> groups = {frozenset({"a", "b"}), frozenset({"c", "d"})}
>>> groups
{frozenset({'a', 'b'}), frozenset({'c', 'd'})}

A frozenset is an immutable version of a set.

[Открыть пост](https://bsky.app/profile/trey.io/post/3mobvsqtoek2p)

----
**🕒 23:19 👤 @trey.io (Trey Hunner)**
💬 Frozen sets support all the same operations (containment checks, set arithmetic) EXCEPT modifications.

>>> frozenset({1, 2, 3}) & frozenset({2, 3, 4})
frozenset({2, 3})

Because a frozenset is immutable, it's hashable, which means it can be used as a set item or a dictionary key.

[Открыть пост](https://bsky.app/profile/trey.io/post/3mobvsqtpds2p)

----
**🕒 23:19 👤 @trey.io (Trey Hunner)**
💬 This often comes up when you need to deduplicate collections of items where order doesn't matter.

For example, finding unique pairs:

unique_pairs = {frozenset({a, b}) for a, b in all_pairs}

For more on what makes something hashable, see my screencast on hashable objects in Python.
