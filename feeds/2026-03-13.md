💬 Python Tip #72 (of 365):

You can redirect all printed output in Python with contextlib.redirect_stdout 🧵

from contextlib import redirect_stdout
import io

with redirect_stdout(io.StringIO()) as output:
    print("hello")

printed_text = output.getvalue()

💬 The redirect_stdout context manager will temporarily set Python's standard output file-like object (sys.stdout) to a different file object. It's most commonly used with an in-memory fake file object (using the io.StringIO class).
