💬 Python Tip #108 (of 365):

Don't use semicolons.

Want to put 2 statements on one line?

from rich.traceback import install; install(show_locals=True)

Don't.

Use 2 lines:

from rich.traceback import install
install(show_locals=True)

It's more readable.

That's the tip. 🤷
