**🕒 18:04 👤 @trey.io (Trey Hunner)**
💬 Python Tip #98 (of 365):

Don't use open method on pathlib.Path objects 🧵

Instead of this:

with path.‍open() as file:
    ...

Prefer this:

with open(path) as file:
    ...

Since Python 3.6, pathlib.Path objects can be passed to the built-in open function.

#Python #DailyPythonTip

💬 That open method in the pathlib.Path class is a relic from an earlier version of pathlib that was not yet fully-embraced by Python's various path-handling features.

I recommend avoiding the old open method and just using Python's built-in open() function everywhere.
