💬 Python Tip #99 (of 365):

Don't convert pathlib.Path objects to strings.

Using a pathlib.Path object in an f-string or print() is fine, but you shouldn't need to str(...) a Path object to pass it off to other code: most path-handling utilities support Path objects.
