💬 Python Tip #79 (of 365):

Don't use strip if you just need to remove a single character 🧵

To remove a single trailing newline, instead of using rstrip:
line = line.rstrip("\n")

Consider using removesuffix:
line = line.removesuffix("\n")

#Python #DailyPythonTip

💬 Using removesuffix() hints that there should only ever be one character removed.

Of course, if our string might have more than one newline character and we'd want to remove all of them, then rstrip would be better.
