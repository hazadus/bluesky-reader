💬 Python Tip #173 (of 365):

Use _ to see the previous REPL result. 🧵

Let's say you ran an expression in the REPL and only decided running it that you want to capture the result:

>>> 2**100
1267650600228229401496703205376

This is what the REPL's magical _ variable is for.
