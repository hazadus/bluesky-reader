**🕒 23:14 👤 @trey.io (Trey Hunner)**
💬 For example, it may not be clear what this checks:

if len(text) - len(text.lstrip()) >= 4: ...

But it's pretty clear what this checks:

indent_level = len(text) - len(text.lstrip())
if indent_level >= 4: ...

Even if a variable is strictly unnecessary, if it improves readability then keep it!
