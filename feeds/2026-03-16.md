**🕒 18:04 👤 @trey.io (Trey Hunner)**
💬 Python Tip #75 (of 365):

Don't use the string split() method for splitting lines.

Instead of this:
lines = poem.split("\n")

Do this:
lines = poem.splitlines()

The string splitlines method will:
• Trim a trailing newline (if there is one)
• Split by "\r\n", "\n", or "\r"
