**🕒 22:45 👤 @trey.io (Trey Hunner)**
💬 Python Tip #175 (of 365):

Pass "-i" to Python to drop into a REPL after your script runs. 🧵

$ python -i fibonacci.‍py
>>> nth_fibonacci(10)
55

You'll have access to everything the script defined.

Passing "-i" is like copy-pasting your entire script into a Python REPL.

#Python #DailyPythonTip

**🕒 22:45 👤 @trey.io (Trey Hunner)**
💬 The script runs first, and then you're dropped into an interactive session with all its variables and functions still in scope, even if the script crashed.

This is handy for debugging: run your script, inspect what's in scope, and figure out what went wrong.
