💬 Python Tip #180 (of 365):

Zero-pad numbers with f-strings. 🧵

Need to format a time like "9:05" instead of "9:5"?
Use "02d" in your f-string format spec.

This week's tips are all related to strings in Python.

#Python #DailyPythonTip

[Открыть пост](https://bsky.app/profile/trey.io/post/3mphnlbc57f2v)

----
**🕒 23:33 👤 @trey.io (Trey Hunner)**
💬 Without zero-padding:

>>> hour, minute = 9, 5
>>> f"{hour}:{minute}"
'9:5'

With zero-padding on the minutes:

>>> f"{hour}:{minute:02d}"
'9:05'

The "02d" format spec pads the number with zeros until it's 2 digits long.

[Открыть пост](https://bsky.app/profile/trey.io/post/3mphnlbf5ty23)

----
**🕒 23:33 👤 @trey.io (Trey Hunner)**
💬 The "0" means "pad with zeros", the "2" is the minimum width, and the d means integer:

>>> n = 37
>>> f"{n:02d}"  # no effect when already 2 digits
'37'
>>> f"{n:04d}"
'0037'

[Открыть пост](https://bsky.app/profile/trey.io/post/3mphnlbhg5i2j)

----
**🕒 23:33 👤 @trey.io (Trey Hunner)**
💬 Zero-padding is also handy for filenames that need to sort correctly:

>>> for n in [1, 2, 10]:
...     print(f"page_{n:02d}.txt")
...
page_01.txt
page_02.txt
page_10.txt

More on zero-padding with f-strings: https://pym.dev/string-formatting/#zero-padding

[Открыть пост](https://bsky.app/profile/trey.io/post/3mphnlcsuhu2u)

**🕒 23:33 👤 @trey.io (Trey Hunner)**
💬 Python's strings also have a zfill method that does something similar:

>>> str(3).zfill(2)
'03'
>>> str(10).zfill(2)
'10'

But zfill requires converting to a string first.

F-string format specs work on numbers.
