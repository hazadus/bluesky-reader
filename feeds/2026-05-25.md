**🕒 18:04 👤 @trey.io (Trey Hunner)**
💬 Python Tip #145 (of 365):

To sort by specific features of an iterable of objects, use a key function 🧵

For example, we could sort strings by their lengths:

>>> colors = ["yellow", "blue", "green", "purple"]
>>> sorted(colors, key=len)
['blue', 'green', 'yellow', 'purple']

#Python #DailyPythonTip

[Открыть пост](https://bsky.app/profile/trey.io/post/3mmp2u6yf6y2r)

----
**🕒 18:04 👤 @trey.io (Trey Hunner)**
💬 A key function can be any callable that can accept each object from the iterable and return a "key" for each.

This all works because functions are objects (see tip 34).

If you wanted to sort objects by multiple characteristics, you could rely on the lexicographical ordering of tuples (see tip 53).

[Открыть пост](https://bsky.app/profile/trey.io/post/3mmp2u76rsa2e)

----
**🕒 18:04 👤 @trey.io (Trey Hunner)**
💬 For example, sorting by length first and then by the characters in the string:

>>> def length_and_value(s): return (len(s), s)
...
>>> sorted(colors, key=length_and_value)
['blue', 'green', 'purple', 'yellow']

Key functions are great for sorting objects *by* one or more features at once.
