ğŸ’¬ Python's "for" loops can only loop in one direction, but reversed() can help you loop in reverse.

for item in reversed(iterable):
    ...

But... why not use slicing?

    ...

Well, that doesn't work for all iterables.

You can only slice sequences.

**ğŸ•’ 19:04 ğŸ‘¤ @trey.io (Trey Hunner)**

If you slice a 1,000,000-item list, you'll make a new 1,000,000-item list.

If you use the reversed function, you'll make a lazy iterable that will loop item-by-item over the original iterable in the reverse direction.
