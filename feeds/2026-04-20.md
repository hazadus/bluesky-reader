**🕒 22:43 👤 @trey.io (Trey Hunner)**
💬 Python Tip #110 (of 365):

Use f-strings instead of the format method when possible. 🧵

Instead of:
statement = "{name}, you have {n} new messages.".format(name=name, n=n)

Do:
statement = f"{name}, you have {n} new messages."

Note that the format method IS still useful.

#Python #DailyPythonTip

**🕒 22:43 👤 @trey.io (Trey Hunner)**
💬 When you need to define your template in one place but use it in another, you the format method can be handy.

For example we might define this at the module-level:
TEMPLATE = "{name}, you have {n} new messages."

And use it later:
statement = TEMPLATE.format(name=user.‍name, n=user.message_count)
