💬 Python Tip #168 (of 365):

Want to start a proper Python REPL within PDB? 🧵

Run the interact command:

(Pdb) interact
*pdb interact start*
>>>

Before Python 3.13 added support for multi-line Python expressions within PDB, I used interact often...
