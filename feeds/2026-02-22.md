**ğŸ•’ 19:04 ğŸ‘¤ @trey.io (Trey Hunner)**
ğŸ’¬ Lexicographical ordering involves comparing item-by-item, starting with the first item in each tuple.

If the first item is different, its value determines the order of the tuples.

If the first item is the same, the next item is compared. If it's the same, the next item is compared, and so on.
