💬 Python Tip #179 (of 365):

Use history & paste mode in the new REPL. 🧵

F2 enters history: previous code is shown WITHOUT >>> prompts, so you can copy just the code.

F3 enters paste mode: Enter won't accidentally run your code block while you're editing it.

💬 Paste mode (F3) is NOT needed for pasting, unless you're running a terminal from the 90's that doesn't support bracketed paste (you don't need to look this up, trust me).

Paste mode (F3) lets you freely edit a code block without accidentally running it.

Press F3 to enter, edit, press F3 to return.
