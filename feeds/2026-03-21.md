**🕒 18:04 👤 @trey.io (Trey Hunner)**
💬 Python Tip #80 (of 365):

Concatenate iterables with the string join method. 🧵

Instead of this:

guests = ""
for name in names:
    if guests:
        guests += ", "
    guests += name

Do this:

guests = ", ".join(names)

#Python #DailyPythonTip

**🕒 18:04 👤 @trey.io (Trey Hunner)**
💬 The join() method doesn't just work on lists. Any iterable of strings will work.

But it MUST be an iterable of STRINGS.

If you have an iterable that isn't strings, you can pair join with a generator expression:

joined = "\n".join(str(n) for n in numbers)

More on this: https://pym.dev/turn-a-list-into-a-string/
