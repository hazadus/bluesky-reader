💬 Python Tip #88 (of 365):

When assigning multiple variables to the same initial immutable value, use multiple assignment

Instead of this:

x = 0
y = 0

You can do this:

x = y = 0

That's safe as long as the value is immutable (numbers, strings, booleans, or None).
