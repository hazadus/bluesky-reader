**🕒 18:04 👤 @trey.io (Trey Hunner)**
💬 Python Tip #102 (of 365):

Use pathlib.Path(directory, filename) to join path strings

To join path strings together, instead of:
joined = Path(directory) / filename

Or:
joined = Path(directory).joinpath(filename)

You can do this:
joined = Path(directory, filename)

#Python #DailyPythonTip

[Открыть пост](https://bsky.app/profile/trey.io/post/3mjcwshagro2o)

----
**🕒 18:04 👤 @trey.io (Trey Hunner)**
💬 Bonus tip: if you don't know whether the path you're joining WITH is a string or a Path object, you can still use the above trick.

The Path class accepts both strings and Path objects.

So this works for Paths:
path / filename

And this works for strings and Paths:
Path(path) / filename
