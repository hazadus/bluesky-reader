**🕒 18:04 👤 @trey.io (Trey Hunner)**
💬 pyupgrade will unneeded Python 2-isms like these:

# coding: utf-8
from __future__ import unicode_literals

It'll also change "super(MyClassName, self)" to super()

And it'll remove unnecessary inheritance from the default object class. "class MyClass(object):" will become "class MyClass:".

[Открыть пост](https://bsky.app/profile/trey.io/post/3mkg5dhydok2t)

----
**🕒 18:04 👤 @trey.io (Trey Hunner)**
💬 pyupgrade is MOSTLY for upgrading from Python 2 idioms to Python 3 ones, but it can also accept arguments like "--py311-plus" to upgrade some older Python 3 idioms to newer ones.

Keep in mind that it won't catch everything and it's not designed to.
